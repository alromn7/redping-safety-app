## üõ†Ô∏è Step-by-Step Stripe Setup Instructions

### Step 1: Access Stripe Dashboard
1. Go to https://dashboard.stripe.com/
2. Sign in to your account
3. Make sure you're in **Test Mode** (toggle in top right) for testing

### Step 2: Create Each Product

For each product above:

1. Click **Products** in left sidebar
2. Click **+ Add product** button
3. Fill in the product details:
   - **Name**: Copy from "Product Name" above
   - **Description**: Copy entire description block above
   - **Image**: (Optional) Upload product image
   - **Statement descriptor**: `REDPING` (appears on credit card)

4. Add **Monthly Price**:
   - Click **Add another price**
   - Price: Enter amount (e.g., 4.99)
   - Billing period: Select **Monthly**
   - Currency: **USD**
   - Click **Add price**
   - **Copy the Price ID** (starts with `price_...`)

5. Add **Yearly Price**:
   - Click **Add another price**
   - Price: Enter amount (e.g., 49.99)
   - Billing period: Select **Yearly**
   - Currency: **USD**
   - Click **Add price**
   - **Copy the Price ID** (starts with `price_...`)

6. Click **Save product**

### Step 3: Record All Price IDs

After creating all products, you'll have 8 price IDs total:

```
Essential+ Monthly: price_________________
Essential+ Yearly:  price_________________

Pro Monthly:        price_________________
Pro Yearly:         price_________________

Ultra Monthly:      price_________________
Ultra Yearly:       price_________________

Family Monthly:     price_________________
Family Yearly:      price_________________
```

### Step 4: Update Cloud Function

Update `functions/src/subscriptionPayments.js` with your new price IDs:

```javascript
const PRICE_IDS = {
  essentialPlus: {
    monthly: 'price_________________', // Your Essential+ monthly price ID
    yearly: 'price_________________',  // Your Essential+ yearly price ID
  },
  pro: {
    monthly: 'price_________________', // Your Pro monthly price ID
    yearly: 'price_________________',  // Your Pro yearly price ID
  },
  ultra: {
    monthly: 'price_________________', // Your Ultra monthly price ID
    yearly: 'price_________________',  // Your Ultra yearly price ID
  },
  family: {
    monthly: 'price_________________', // Your Family monthly price ID
    yearly: 'price_________________',  // Your Family yearly price ID
  },
};
```

### Step 5: Deploy Updated Function

```bash
cd functions
firebase deploy --only functions:processSubscriptionPayment
```
