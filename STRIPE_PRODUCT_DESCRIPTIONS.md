## üí° **Tips for Stripe Setup**

1. **Product Type**: Select "Service" (not "Good")
2. **Recurring**: Set billing to "Monthly" or "Yearly"
3. **Tax Code**: Use "Software as a Service (SaaS)" tax code
4. **Statement Descriptor**: Use "REDPING SAFETY" (appears on credit card statements)
5. **Metadata**: Add tier name, features, and limits for easy tracking

### Metadata Examples:
```json
{
  "tier": "essential_plus",
  "features": "acfd,medical_profile,hazard_alerts,sos_sms",
  "emergency_contacts": "5",
```
