# RedPing SMS Calling Card Design

## Overview
The RedPing SMS emergency notification system now features a professional **calling card style** design with distinctive @REDP!NG branding. Each SMS message is formatted as a visually striking, easy-to-read emergency alert card.

## Design Philosophy

### Visual Identity
- **Header Box**: Consistent `@REDP!NG` branding with decorative box characters
- **Section Dividers**: Clean horizontal lines (â”) for visual organization
- **Professional Layout**: Corporate calling card aesthetic with structured sections
- **High Contrast**: Critical information stands out immediately
- **Brand Recognition**: Instant identification as RedPing emergency alert

### Key Features
1. **Unique Branding**: `@REDP!NG` logo header in every message
2. **Visual Hierarchy**: Box characters and dividers organize information
3. **Clickable Links**: Phone numbers and URLs on separate lines
4. **Action-Oriented**: Clear numbered steps for emergency response
5. **Professional Appearance**: Like receiving a business card by SMS

## Legal & Compliance

### SMS Content Regulations
- **TCPA Compliance**: Emergency messages exempt from opt-in requirements
- **Character Safety**: No characters that could be interpreted as commands
- **Link Safety**: All URLs are HTTPS or valid app deep links
- **Privacy**: No sensitive medical data in SMS body

### Accessibility
- **Screen Readers**: Emoji have standard descriptions
- **Color Blind**: Uses symbols, not color alone
- **Dyslexia-Friendly**: Short lines, clear spacing
- **International**: Unicode characters work globally
